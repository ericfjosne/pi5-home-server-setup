# Raspberry Pi 5 setup as Home Server

## Description

This guide describes how to configure a Raspberry Pi 5 as a tiny yet mighty home server. It can also probably be used for earlier Raspberry Pi devices, or even regular computers.

It is best experienced as a Github page, which is available over here: [https://ericfjosne.github.io/pi5-home-server/](https://ericfjosne.github.io/pi5-home-server/)
