---
title: Report an issue
---

If you see something wrong in this guide, or have a suggestion on how to improve it, I would love to hear about it!

This website offers 2 ways to report issues/provide feedback:
- using the _report issue_ link found at the top of each page (under its title).
- selecting the text for which you would like to report an issue, right clicking on the selection, and opening the _report issue_ pop-up link which appears.

Please provide as much context as you can and I'll make sure to address your feedback when I can!
